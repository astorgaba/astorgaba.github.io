PS5 engine, Deferred Rendering, Snake Game

Project Date Sep. 2023 - May. 2024
Features C++, PS5 low level API

This project is part of the course at Sheffield Hallam University. It was divided into 2 semesters. The first one was focused on working on an engine by ourselves and to develop some interesting techniques, like deferred rendering, which was the one that I chose to develop.

The second semester was focused on building a game with other students from the university. I worked along 3 other programmers and 2 designers. The features that I developed on this project were the addition of the physics library that I developed on the first semester of this same year while adding more features to it, like a way to create the "snake" by keeping some bodies constrainted or collision events, creating the adaptative snake mesh and the UI.
